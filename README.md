# vietnam-provinces

## Install

```bash
npm install vietnam-provinces
```

## Use

### CommonJs
```bash
const { getProvincesWithDetail } = require('vietnam-provinces');

(() => {
  console.log(getProvincesWithDetail());
})();
```

### ES Module

```bash
import pkg from 'vietnam-provinces';
const { getProvincesWithDetail } = pkg;

(() => {
  console.log(getProvincesWithDetail());
})();
```
